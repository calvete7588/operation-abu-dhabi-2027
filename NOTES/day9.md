## Day 9 – Security Logs, SIEM Dashboards, Splunk & Chronicle (Course 2 - Module 3)
**Date:** July 21, 2026

## Objective
Analyze log types (firewall, network, server), examine SIEM dashboard visualization techniques, and compare industry-standard SIEM platforms like Splunk and Chronicle.

## Core Concepts Explained
* **Log Analysis:** Investigating raw system logs (firewalls, servers, network devices) to detect unauthorized activities and trace attack vectors.
* **SIEM Dashboards:** Visual representations of real-time security events that allow SOC teams to quickly assess security posture and prioritize alerts.
* **Enterprise SIEM Solutions:**
  * *Splunk:* Market-leading platform using SPL for deep log analysis and custom alerting.
  * *Chronicle (Google Cloud):* Cloud-native SIEM designed for massive-scale data ingestion and rapid threat detection.

## Technical Vocabulary
* **Log Parsing** (/log par-sing/): Procesamiento e interpretación de registros de eventos.
* **Dashboard** (/dash-bord/): Panel visual de control y monitoreo.
* **SPL** (/es-pi-el/): Search Processing Language (Lenguaje de consulta de Splunk).
